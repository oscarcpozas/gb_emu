Trying to implement a GameBoy emulator in Rust. The goal is to have [100% accuracy with the original hardware](https://mgba.io/2017/04/30/emulation-accuracy).

### How to run the emu?

```bash
cargo run --bin emu -- ./misc/tetris.gb
```

### How serve the book?

```bash
mdbook serve book
```
